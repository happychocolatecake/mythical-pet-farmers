# Mythical Pet Farmers

### Summary
Mythical Pet Farmers is a 2D unity game that combines farming simulation with RPG-style pet battles. Players can choose to focus on growing crops, battling with their pet, or both. The game offers freedom with no time limits, allowing for a relaxed or strategic playstyle. Inspired by games like Pok√©mon and Stardew Valley, it features simple controls, a day-night cycle, and a save system. Players can select from different pets, engage in battles using a button-mashing system, and improve their farm over time. Designed for casual gamers, it provides an accessible and engaging experience.
